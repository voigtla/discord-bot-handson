# 3️⃣ README.md を作る（本日のメイン作業）

ここからが **この回のハンズオン本体**です。

---

## 3-1. README.md を作成する

### いま開いている画面：VS Code（エクスプローラー）

- `git_practice` を右クリック
- **新しいファイル**
- ファイル名：`README.md`

---

## 3-2. README.md（全文）

**以下を、そのまま丸ごと貼ってください。**

```md
# Discord Bot（ハンズオン成果物）

このリポジトリは、ハンズオン形式で作成した
**最小構成の Discord Bot** です。

---

## このBotがすること

- `/hello`
  - 実行すると記録を1件保存する
- `/count`
  - 自分の記録件数を数字で返す

※ 評価・助言・分析は行いません

---

## 必要なもの

- Node.js
- Discord アカウント
- Discord Bot（作成済み）
- 環境変数を設定した `.env` ファイル

---

## フォルダ構成

```text
git_practice/
├─ index.js        # Bot本体
├─ register.js     # コマンド登録用（必要なときだけ使う）
├─ aiFormatter.js  # 文章整形（意味は変えない）
├─ responses.js    # Botが返してよい文章
├─ safeReply.js    # エラー時の安全な返信
├─ data.db         # SQLite（記録用）
├─ .env            # 秘密情報（Gitに含めない）
├─ .gitignore
├─ README.md
└─ node_modules/
````

---

## 使い方（最短手順）

このリポジトリは、すでに必要な準備が完了している前提で使います。

### Botを起動する

```bash
node index.js
```

---

## 動作確認

Discordで以下を実行します。

* `/hello`
* `/count`

数字が返れば正常です。

---

## うまく動かないとき

* Botが起動しているか
* `.env` が設定されているか
* `node index.js` を実行したフォルダが正しいか
* ターミナルにエラーが出ていないか

※ `.env` の中身は共有しないでください

---

## このBotがやらないこと

* 行動の評価
* 助言・提案
* 感情の解釈
* ユーザー入力のAI送信

````
